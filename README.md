# Glow Shop

A real-time Korean skincare e-commerce backend built to learn full-stack engineering end to end.

## Stack
- **Frontend**: React + Vite + Tailwind
- **Gateway**: Node.js + Express
- **Services**: Order, Inventory, Payment (Node.js microservices)
- **Messaging**: Apache Kafka
- **Databases**: PostgreSQL + Redis
- **Infrastructure**: Kubernetes (minikube)
- **CI/CD**: GitHub Actions
